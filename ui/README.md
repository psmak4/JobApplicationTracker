# Job Application Tracker - Frontend

This is the React frontend for the Job Application Tracker, built with:

- **React 19**
- **Vite**
- **Tailwind CSS 4**
- **shadcn/ui** components
- **TanStack Query**
- **React Router 7**

## Key Architecture

- **Routing:** Split views for `Dashboard` (`/app`), `Pipeline` kanban (`/app/pipeline`), and `Archive` (`/app/archive`).
- **Navigation:** Top-level navigation using `SiteHeader`.
- **Design:** Utilizing Design System 2.0 with refined colors, rings, and status badges.

## Development

Run `npm run dev` to start the development server at http://localhost:5173.
The application dashboard is available at `/app`.
